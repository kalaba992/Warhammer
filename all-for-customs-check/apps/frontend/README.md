# Frontend

React 19/TypeScript app using shadcn, TailwindCSS 4, Radix, React Router v7.

## Setup
- Install dependencies: `npm install`
- Start dev server: `npm run dev`

## Features
- Multi-tenant RBAC/ABAC
- Auth0 OIDC, FIDO2, MFA
- Document upload & processing
- RAG search UI
- Evidence Bundle viewer
- Stripe/email/SMS/OpenAI integrations
- Strict security & EU compliance
