# Backend

Convex backend with Node.js runtime for AI helpers, document processing, and secure API.

## Setup
- Install dependencies: `npm install`
- Start backend: `npm run dev`

## Features
- REST API v1 (OpenAPI v3)
- Contract tests
- Document processing (PDF/Excel/OCR)
- RAG vector store
- Evidence Bundle schema & signing (HSM/KMS stub)
- Auth0 OIDC, RBAC/ABAC
- Integrations: Stripe, email, SMS, OpenAI
- Observability, security, EU compliance
